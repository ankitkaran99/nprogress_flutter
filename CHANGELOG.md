## 0.0.1

- Initial release.
- Added `NProgress` global singleton facade (`start`, `set`, `inc`, `done`, `remove`, `configure`).
- Added `NProgressController` for state management.
- Added `NProgressWidget` and `NProgressOverlay` (`NProgress.builder()`).
- Added `NProgressNavigatorObserver` for automatic navigation route tracking.
- Support for custom colors, gradients, glowing tip (peg) effect, spinner, and trickling.
